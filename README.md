## 项目概述

NeoMap 前端是一个多类型地图发布平台，主要功能包括：

- 多类型地图渲染
- 空间分析工具
- 用户标注与测量工具
- 多图层管理
- 地图——群组 管理系统

## 核心技术栈

- **框架**: Next.js 15.4.1 (App Router)
- **React**: 19.1.0
- **框架**: Turbopack
- **UI 组件**: Tailwind CSS
